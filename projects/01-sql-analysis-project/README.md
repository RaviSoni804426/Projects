# SQL Analysis Project: Sales Data

This project demonstrates SQL proficiency by creating a schema, inserting sample data, and performing analytical queries.

## Structure
- `schema.sql`: Database schema definition and data insertion.
- `queries.sql`: Analytical queries for revenue, trends, and category performance.

## How to Use
You can run these scripts using any SQLite or MySQL client. 
For SQLite (terminal):
```bash
sqlite3 sales_analysis.db < schema.sql
sqlite3 sales_analysis.db < queries.sql
```

## Insights Provided
1. Total revenue per product.
2. Monthly sales trends.
3. Performance by category.
